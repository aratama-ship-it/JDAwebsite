# トップページ デザイン改善メモ

作成日: 2026-07-08
対象: `index.html` / `css/style.css` / `js/main.js`

## 確認条件

- ローカル確認: `http://127.0.0.1:8000/index.html`
- 確認幅: 1440x900 / 768x1024 / 390x844
- 方針: 機能追加より、トップページの見た目・余白・第一印象を優先する。

## 現状の強み

- ヒーローの写真、暗いオーバーレイ、太い見出しの組み合わせはかなり強く、競技サイトらしい迫力がある。
- 白背景に赤アクセントの方向性は、協会公式サイトとして見やすい。
- PICK UP、CHAMPIONS、UPDATES の並びは情報設計として自然。
- スマホ幅では `midd` 帯が非表示になり、トップからPICK UPまでの距離が短くなるため読み進めやすい。
- 現行の `images/` に残した素材はすべて参照中。未使用素材は `archive/unused-assets/` に退避済み。

## 優先度A: まず直したい

1. **アンカー遷移時の見出し被りを直す**
   - `#athletes` や `#updates` に直接移動した時、固定ヘッダーの下にセクション見出しが近すぎる、または一部隠れたように見える。
   - `scroll-margin-top` を `.section` や対象IDに設定すると改善できそう。
   - 実装済み(`style.css?v=56`): `#news` / `#athletes` / `#updates` に `scroll-margin-top` を設定。PCは96px、700px以下は88px。

2. **スマホのCHAMPIONSカード内テキストを読みやすくする**
   - 390px幅ではカードが横スクロールで見えて良いが、選手名・部門名が写真の上でやや詰まり気味。
   - テキスト背景のグラデーションを少し強める、文字サイズを微調整する、またはカード幅をわずかに広げる候補。
   - 実装済み(`style.css?v=56`): 560px以下で `#athletes` のカードだけオーバーレイを強め、名前/部門名の文字サイズ・行間・テキスト影を調整。カード幅は変更なし。

3. **PICK UPへの導線を少し軽くする**
   - デスクトップではヒーロー、マーキー、`midd`帯が連続し、PICK UPの開始がやや下に押し出される。
   - `midd`帯は魅力があるので削らず、表示高さやクロップ位置を少しだけ調整する方向が良さそう。
   - 実装済み(`style.css?v=56`): `.section-divider` を135px→112px、画像固定高さを560px→500px、上方向のずらしを10px→8pxに調整。スマホ非表示は維持。

## 優先度B: 磨くと完成度が上がる

1. **ヒーローの文字位置をスライドごとに点検する**
   - 写真の主役位置によって、左側テキストとのバランスが変わる。
   - 現在は全スライド共通配置なので、必要ならスライドごとに微調整用クラスを持たせる。
   - 実装済み(`style.css?v=56`): `hero-slide--oidc` / `hero-slide--tar` / `hero-slide--regu` / `hero-slide--access` を追加し、テキスト位置・最大幅・画像位置・グラデーションをスライドごとに調整。

2. **UPDATESのカード表現を整える**
   - 最新項目のピンク背景は目立って良いが、他項目との差がやや大きい。
   - 最新項目はそのまま、通常項目に薄い境界線や hover の統一感を加えると、公式サイトらしい落ち着きが出そう。
   - 実装済み(`style.css?v=56`): `.rule-item:has(.update-text)` に限定して、更新カードを白背景・薄い境界線・軽い影・浮くhoverへ調整。最新項目は赤のグラデーションを少し控えめに維持。

3. **スマホのヘッダーでブランド名をどう扱うか検討する**
   - 390px幅ではロゴのみになり、すっきりしている一方で初見では協会名が少し弱い。
   - ロゴだけでよいか、短縮表記 `JDA` を出すかは好みの判断。
   - 実装済み(`style.css?v=56`): スマホではロゴ横に `JDA` を表示し、768px以上では正式名称表示に切り替え。480px以下はヘッダー余白も少し調整。

4. **セクション見出しの日本語サブテキストの見え方を揃える**
   - `PICK UP / ピックアップ`、`CHAMPIONS / 日本チャンピオン`、`UPDATES / 更新情報` は良い構成。
   - スマホではサブテキストの余白・濃さを少し揃えるとより綺麗。
   - 実装済み(`style.css?v=56`): `.section-jp` を赤いピル型ラベルにして、PC/スマホで余白・濃さ・サイズ感を統一。

## 優先度C: モダン寄りの追加候補

1. **控えめなスクロールイン演出**
   - PICK UPカード、CHAMPIONS、UPDATESをふわっと表示する程度。
   - やりすぎると公式感が落ちるので、`opacity` と `translateY(12px)` 程度に留める。

2. **`prefers-reduced-motion` 対応**
   - 既にカルーセルやマーキーが動くため、モーションを抑えたいユーザー向けのCSSを用意すると今っぽい。
   - デザイン品質とアクセシビリティの両方に効く。

3. **カードのフォーカス状態を整える**
   - キーボード操作時にカードやボタンのフォーカスが見えやすいと、細部の品質が上がる。

## 次に実装するなら

最初の一手は、以下の小さなCSS改善がよさそう。

1. 完了: `#news` / `#athletes` / `#updates` の `scroll-margin-top` 調整。
2. 完了: スマホCHAMPIONSカードのオーバーレイと文字サイズ調整。
3. 完了: デスクトップの `midd`帯の高さと見せ位置を微調整。

この3つは見た目への効果が大きく、機能面のリスクが小さい。
