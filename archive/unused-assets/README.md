# unused-assets archive

このフォルダは、現行サイトから参照されていない旧素材・元素材の保管場所です。

削除ではなく退避です。将来再利用する場合は、必要なファイルを元のパスへ戻すか、HTML/CSS/JS/JSON側の参照パスを更新してください。

## 退避した主な内容

- `images/champions/<年度>/*.png`: WebP化後の元PNG。
- `images/championsath/`: 旧CHAMPIONS用のダミー/試作画像。
- `images/events/`: 現在未使用の大会バナー画像。
- `images/hero/`: 現在未使用の旧ヒーロー/CTA画像。
- `images/logo/`: 旧ロゴ、未使用フルロゴ、`midd.png` の元PNG/PSD。
- `images/pickup/*.png` と `final_news_*.jpg`: WebP化後の元PNGと旧PICK UP画像。
- `images/poker/cardback.png`: WebP化後の元PNG。
- `root/`: ルート直下にあった旧 `midd.png` / `middold.png`。
