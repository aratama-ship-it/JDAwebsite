# OIDC 多言語編集マニュアル

`external/OIDC/index.html` は、同じページ内で日本語・英語・中国語簡体字・中国語繁体字を切り替える構成です。ヘッダー右端の `JP / EN / 简 / 繁` を押すと、ページ遷移なしで表示文言が切り替わります。

## 仕組み

- HTML内の翻訳対象には `data-i18n="キー名"` または `data-i18n-html="キー名"` が付いています。
- ページ最下部の `const translations` に、`ja` / `en` / `zh-CN` / `zh-TW` の4言語分をまとめています。
- 選択した言語はブラウザに保存され、次回の閲覧時にも同じ言語で表示されます。
- タイトル・説明文・`<html lang>` も切り替わります。

## 文言を更新する手順

大会名、日程、更新情報などを変更する時は、次の順に編集します。

1. `external/OIDC/index.html` の通常表示（日本語）を更新する。
2. 同ファイル末尾の `translations` で、同じキーを4言語すべて更新する。
3. 改行やリンクを含む文言は `data-i18n-html` を使う。リンク先は各言語の文言内でも同じURLを保つ。
4. `data-i18n` のキーを追加・変更した場合は、4言語すべてに同名キーがあることを確認する。
5. ローカルサーバーで `JP / EN / 简 / 繁` を1回ずつ押して、レイアウトとリンクを確認する。

## 注意点

- `data-i18n-html` は `innerHTML` で差し替えます。外部からコピーした未確認のHTMLをそのまま入れないでください。現在は制作者が管理する翻訳文とリンクだけを入れる前提です。
- 登録ページとPDFは現在の `diabolo.jp/OIDC/` の既存URLを開きます。これら外部ページやPDFそのものは、この切替では翻訳されません。中国語版の登録フォーム・要項を作る場合は、各言語のURLを用意してから翻訳文内のリンク先も分けます。
- 大会固有の用語は、簡体字では「空竹」、繁体字では「扯鈴」としています。大会として別の公式表記を採用する場合は、該当する中国語の翻訳を一括で見直してください。
- ヘッダーの言語ボタンは短い表記のままにしています。`简` は簡体字、`繁` は繁体字です。
